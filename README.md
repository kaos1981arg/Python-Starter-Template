## 🛠️ Stack Tecnológico
El proyecto utiliza las herramientas más modernas del ecosistema Python:

* **Gestor de paquetes:** [uv](https://github.com/astral-sh/uv) (Extremadamente rápido).
* **Linter & Formatter:** [Ruff](https://beta.ruff.rs/docs/) (Sustituye a Black, Isort y Flake8).
* **Type Checking:** [Mypy](https://mypy-lang.org/) (Tipado estático).
* **Testing:** [Pytest](https://docs.pytest.org/).
* **Documentación:** [MkDocs](https://www.mkdocs.org/) con el tema Material.
